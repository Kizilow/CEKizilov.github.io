🚀 Как запустить на GitHub Pages
Бесплатный хостинг — открывается с любого устройства, телефона, ноутбука, планшета
1
Создай аккаунт на GitHub
Зайди на github.com → Sign up (бесплатно). Придумай короткий username, например shipeng-vasyl
2
Создай новый репозиторий
Нажми + → New repository. Назови его точно так:
ТвойUsername.github.io
Например: shipeng-vasyl.github.io
Поставь галочку Public → Create repository
3
Загрузи все 6 файлов
Нажми uploading an existing file (или перетащи файлы). Загрузи все сразу:
index.html
fuel-consumption-calculator-4-3.html
pms-notes-5.html
ACCOMMULATOR-CHECK-Pro.html
Contract-Tracker-Pro-5-2.html
LOG-BOOK-4.html
Нажми Commit changes
4
Включи GitHub Pages
Settings → Pages → Source: выбери main branch → Save
Подожди 1–2 минуты.
5
Готово! Открывай с любого устройства
Твой сайт будет доступен по адресу:
https://ТвойUsername.github.io
Сохрани ссылку в телефон — работает и без интернета после первой загрузки.
💡 Важно: файл стартовой страницы должен называться именно index.html — GitHub Pages автоматически открывает его первым. Все остальные файлы должны лежать в той же папке (корне репозитория).
